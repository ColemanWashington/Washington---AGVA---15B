What To Start Out With

There are mutiple of different things that you should start off with, but there is somethings that I recommend.

You should have ready:

  • A 3d modelling website

For beginners I suggest tinkercad.com

  • A folder ready - optional
  • a website for changing the code

It is just helpful to be able to set the correct settings so that the print won't be messy or mess up when you run it on the printer. Also you want to convert the file type to .gcode

If you are using Tinkercad then that's good, because tinkercad gives you a wide variety of blocks to work with and it basically set up for the use of begginers. If you have used a 3d moddeling software before and prefer to use it then thats good too.

How To Export the file Into a 3D Printer Software

So after you have finally finished with making the thing you wanted to 3d print in tinkercad, or the editor software you used, you would want to find somewhere on the screen that says export. For tinkercad that would be in the top right. After you have exported your file you would want to go over to the software kiri:moto. Like before that was just a suggestion from me, but you can go to one that you are familiar with. Once you are in kiri:moto, you would want to change your printer type to the printer you have, or are going to use. The way to do this in kiri:moto is by heading to the gear icon in the top right, then click on machines go the the drop down menu next to select device, then search for your device. After you have done so you would want to change the settings on the left side.

Change the following:

  • Base
  • Cooling
  • Sparse Fill
  • Output
  • Expert

After you have edited all of those to the best setting they have, make sure you look them up, you would want to finally export it

Sending the File to the Printer

To send the file that you finally exported to the printer, you would want to take the .gcode file and put it into a folder. You are putting them into folders so you can group all of your codes together so you won't just go looking everywhere for the file, and eventually giving up and re-exporting the file. Once you have succesfully put the file into the folder you would want to get your USB drive that was listed before so you would be able to put the file in there. So you would want to plug your USB drive in the USB slot and locate it in the files app. Open the folder in that you put your file in, and then drag and drop it into the usb slot. After you have dropped it in the USB, eject the USB by clicking the eject button. Finally plug the USB into the pinter and start printing it. Make sure you check on it every hour if it has a long print session.
